Technical details

The project website "America's First Petrochemical Corridor" was created with CollectionBuilder (CB-CSV), a Jekyll-based open source framework for creating digital collection and exhibit websites.

Maps were created using X, Y, Z tool and LeafletJS.

The source code can be found on GitHub and is licensed under CC BY-NC 4.0.
